Workshop Content:

  Core application:
  - agent_harness.py — importable agent module with 4 workshop placeholders (CUSTOM_TOOLS, guardrail, skill loader,
  standalone entry point)

  - templates/index.html — single-page UI: file browser, drag-and-drop upload, log terminal, model selector
  - requirements.txt + Containerfile (RH UBI 9 Python 3.12, port 8080) + run.sh

  Sample inbox:
  - inbox/instruction.md — working CSV analysis task
  - inbox/sample_data.csv — employee data to analyse

  Documentation:
  - README.md — full project docs with architecture diagram and API reference
  - Quickstart.md — 5-minute getting-started guide
